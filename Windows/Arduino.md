# Arduino

## 下载Arduino IDE

打开网页输入网址: http://arduino.cc/en/Main/Software

## 设为中文界面

选择菜单栏File → Preferences，选择Editor language → 简体中文，点击OK。

关闭Arduino IDE，重新打开，就是中文界面。

## 配置IDE

打开Arduino→文件→首选项

在附加开发板管理器网址中添加
